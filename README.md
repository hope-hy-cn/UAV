### 测试

1. 方式一：选择文件
      - 点击选择文件，进入文件夹选择 <测试数据.text>
2. 方式二：手动输入内容
      - eg: 
      ```
      Plane1 1 1 1
      Plane1 1 1 1 1 2 3
      Plane1 2 3 4 1 1 1
      Plane1 3 4 5 
      Plane1 1 1 1 1 2 3
      ```

> 以上面5条数据为例
>
